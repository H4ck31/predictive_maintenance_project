# Проект: Бинарная классификация для предиктивного обслуживания оборудования

## Описание проекта
Цель проекта – разработать модель машинного обучения, которая предсказывает отказ оборудования.

## Датасет
Используется датасет **"AI4I 2020 Predictive Maintenance Dataset"** (UCI Repository, id=601).
Содержит 10 000 записей с 14 признаками.

## Установка и запуск

1. Клонируйте репозиторий или скачайте папку проекта

   git clone https://github.com/H4ck31/predictive_maintenance_project
   
2. Установите зависимости:

 pip install -r requirements.txt

3. Запустите приложение:

 streamlit run app.py

Или просто дважды кликните по файлу `run_app.bat`

## Структура проекта
- `app.py` - Основной файл приложения
- `analysis_and_model.py` - Страница с анализом данных и моделью
- `presentation.py` - Страница с презентацией
- `requirements.txt` - Зависимости проекта
- `run_app.bat` - Файл для быстрого запуска
- `data/` - Папка для данных
- `video/` - Папка с видео-демонстрацией

## Результаты
Наилучшая модель: Random Forest
- Accuracy: ~0.99
- ROC-AUC: ~0.99
